## About

Boilerplate code for making bots using Distype. 

## Installation

```sh
npm i @br88c/distype-boilerplate
```

### Prerequisites

- **[Node.js >=16.13.0](https://nodejs.org/)**
- **[NPM >=8.1.0](https://www.npmjs.com/)**
